Reports fields annotated with @ConditionalSynced that do not have a valid corresponding evaluation method.

The evaluation method must meet the following requirements:

  • The method name must match the methodName attribute of the annotation.
  • The method must return boolean.
  • The method must have exactly one parameter, and its type must be compatible with the field's type.

Example:


@ConditionalSynced(methodName = "shouldSync")
int field = 10;

public boolean shouldSync(int value) {
    return value > 0;
}
